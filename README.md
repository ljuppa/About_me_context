# About Me — AI Context File

This repository contains my personal AI context file: a structured calibration document I load at the start of every AI session to eliminate setup friction and get consistently useful output across tools.

## What it is

Most AI interactions start with the same invisible tax — explaining who you are, how you think, what format you want, what you don't want. This file pays that tax once and removes it from every session.

It covers:
- **Role and domain calibration** — who I am, what I work on, what level to pitch responses at
- **AI interaction rules** — one question at a time, no sycophancy, present options not decisions
- **Three AI roles** with task examples and a trigger guide — Thinking Partner, Executor, Coach
- **Writing voice** — style rules, voice patterns, and a confirmed sample paragraph for when AI writes in my voice
- **Failure modes** — how the AI self-corrects without being asked
- **Escalation signals** — when to stop and flag rather than proceed
- **Behavioral rules** — operationalised values, not labels
- **Portable Core** — 12 rules that work in any tool that doesn't support full context files

## How to use it

**Claude:** Create a Project at claude.ai, paste the full file as project instructions. Loads automatically every session.

**Gemini:** Create a Gem, paste the full file as the system instructions.

**ChatGPT:** Create a Custom GPT or paste into Custom Instructions.

**Copilot / other tools:** Use the Portable Core section at the bottom of the file — 12 rules that fit any system prompt field.

## How to adapt it for yourself

The structure is the reusable part. Replace the content with your own:
1. Role Calibration — your title, team size, domain
2. Current State — what you're working on right now, live tensions
3. Writing Voice — your style rules and a paragraph you've written that felt right
4. AI Roles — the three modes work for most people; adjust the task examples
5. Portable Core — update rules 11 and 12 to match your voice and domain

The file was built iteratively through a judging panel process (best friend, AI engineering expert, senior VP, life coach, prompt engineer, model behaviour researcher, daily AI user). Final panel score: **9.8/10**.

## Maintenance

Update **Current State** and **Tools in Active Use** when something materially changes. Full file review quarterly.
